・改築時に過去の失敗を帳消しにできる。

Core apps

apeman has several build-in prototype apps.

You can create an app in a minute by inheriting one of them.

They comes with a lot of features and automation tasks,

but don't have to use all of then and can cut them off easily.

The most basic concept of is "write anyway, judge later".

Abstract

Apeman most basic app.

All apps begins from here.

Contains basic tasks like install, build, test, doc, and so on.

All features are extensible, and more over, overridable.

Static

App to serve static files.

A child of "Abstract".

Contains clustering, multi-localing, compiling.

Depends on connect middleware modules.

Restful

Restful web api app.

A child of "Static".

Contains validation, format, dynamic rooting, data-eternalize.

Depends on express modules.

Authenticable

Authenticable web app.

A child of "Restful".

Contains signing, authorizing, role filtering.

Client side pages are depends on angular.

Social

Social web app.

A child of "Authenticable".

Contains oauth with major social web sites.
